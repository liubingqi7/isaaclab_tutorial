`具身智能仿真实战：构建、训练与部署.pdf`.

The local directory is for notes, manifests, source maps, and validation
records. Primary source code and compute live on `liubingqi@192.168.2.16`,
using conda env `env_isaaclab_2.3_lbq`.

Start with:

- `AGENTS.md` for operating rules.
- `workspace_manifest.yaml` for remote paths, upstream sources, and current
  validation state.
